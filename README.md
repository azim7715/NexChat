# NexChat

NexChat is a real-time chat application built using the MERN stack. It supports one-to-one messaging, group chats, and live updates using Socket.io.

I built this project to practice full-stack development and understand how real-time communication works in web applications.

---

## Features

* Real-time messaging using Socket.io
* One-to-one chat and group chat
* Online/offline user status
* Typing indicators
* Image sharing
* Message reactions
* Read receipts
* OTP-based signup (development mode)

---

## Tech Stack

Frontend:

* React (Vite)
* Tailwind CSS
* Zustand

Backend:

* Node.js
* Express.js
* MongoDB (Mongoose)
* Socket.io

## Run Locally

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev
